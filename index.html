¿Qué son los Componentes Pasivos en la Red?

Los componentes pasivos en una red son aquellos que no requieren energía eléctrica para funcionar. Se encargan de transmitir datos y señales a través de la infraestructura de red sin procesarlos ni amplificarlos.

RJ45

El conector RJ45 es un conector estándar para cables de red Ethernet. Es utilizado para conectar dispositivos de red a través de cables de par trenzado (UTP).

Jacks o Keystone

Los jacks o keystones son conectores modulares que se insertan en placas de pared o paneles de parcheo, permitiendo la conexión de cables UTP o de fibra óptica.

Cable UTP

El cable UTP (Unshielded Twisted Pair) es un tipo de cable de par trenzado sin blindaje utilizado en redes Ethernet para la transmisión de datos.

Patch Panel

El patch panel es un dispositivo que organiza y distribuye cables de red en una infraestructura, permitiendo interconectar dispositivos de red de forma ordenada.

Patch Cord

El patch cord es un cable corto con conectores RJ45 en ambos extremos, utilizado para conectar dispositivos de red dentro de un rack o gabinete.

Fibra Óptica

La fibra óptica es un medio de transmisión que utiliza luz para transportar datos a alta velocidad en largas distancias, siendo inmune a interferencias electromagnéticas.

Canaletas

Las canaletas son conductos que protegen y organizan cables de red en instalaciones, evitando daños físicos y facilitando el mantenimiento.

¿Qué son los Componentes Activos en la Red?

Los componentes activos en una red son aquellos que requieren energía eléctrica para funcionar. Se encargan de procesar, amplificar y dirigir el tráfico de datos dentro de la red.

Switch

El switch es un dispositivo que conecta múltiples dispositivos en una red local (LAN) y permite la transferencia de datos entre ellos de manera eficiente.

Router

El router es un dispositivo que conecta redes diferentes, como una red local a Internet, y dirige el tráfico de datos entre ellas.

Tarjeta de Red

La tarjeta de red es un componente que permite a un dispositivo conectarse a una red, facilitando la transmisión y recepción de datos.

Interruptor Inteligente Wifi

El interruptor inteligente Wifi permite controlar dispositivos eléctricos de forma remota a través de una red Wifi.

Access Point

El Access Point es un dispositivo que extiende la cobertura de una red inalámbrica, permitiendo que más dispositivos se conecten a ella.

Herramientas para Cableado Estructurado

Estas son las herramientas más comunes utilizadas para realizar instalaciones de cableado estructurado en redes.

Ponchadora

La ponchadora es una herramienta utilizada para conectar cables de red en jacks o patch panels, asegurando un buen contacto entre el cable y el conector.

Crimpadora

La crimpadora es una herramienta que se utiliza para colocar conectores RJ45 en los extremos de los cables de red, asegurando una conexión firme.

Desforradora

La desforradora se utiliza para quitar el recubrimiento externo de los cables de red sin dañar los conductores internos.

Tester / Probador de Cables

El tester de cables es una herramienta que verifica la continuidad y el correcto funcionamiento de los cables de red, asegurando que no haya fallos en la instalación.

Cableado Horizontal y Vertical

Cableado Horizontal

El cableado horizontal se refiere a los cables que conectan las estaciones de trabajo a los puntos de distribución en una misma planta de un edificio.

Cableado Vertical o Backbone

El cableado vertical o backbone es el conjunto de cables que conectan diferentes plantas de un edificio, generalmente entre los diferentes cuartos de telecomunicaciones.

Tipos de Cables para Cableado Horizontal

Cable UTP

El cable UTP (Unshielded Twisted Pair) es el más común en redes LAN. Tiene pares de hilos trenzados sin blindaje para reducir interferencias.

Cable FTP

El cable FTP (Foiled Twisted Pair) es similar al UTP pero incluye una lámina metálica que reduce la interferencia externa.

Cable SFTP

El cable SFTP (Shielded Foiled Twisted Pair) tiene tanto trenzado como blindaje alrededor de los pares de cables, ofreciendo mayor protección contra interferencias.

Normativa para Cables de Red (Clase A y B)

La normativa de cables de red define los estándares de conexión en las instalaciones. Las configuraciones Clase A y Clase B establecen diferentes órdenes de los pares de cables en los conectores RJ45.

Usos de Cables en la Red

Cable Directo

El cable directo se utiliza para conectar dispositivos diferentes, como una computadora a un switch o router.

Cable Cruzado

El cable cruzado se utiliza para conectar dispositivos similares entre sí, como dos computadoras directamente o dos switches.
